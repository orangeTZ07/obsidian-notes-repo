# 转动惯量核心公式

## 1. 核心定理
- **[[平行轴定理]]**: $I = I_c + md^2$ 
  *(用于已知质心转动惯量，求平移轴转动惯量)*
- **[[垂直轴定理]]**: $I_z = I_x + I_y$ 
  *(仅适用于薄板)*

## 2. 常见刚体的转动惯量 ($I$)
> [!NOTE] 记忆重点
> 质量为 $m$、几何尺寸如下的均质刚体公式：

### 细杆 (长 $L$)
*   转轴过**中心**并垂直于杆：$I = \frac{1}{12}mL^2$
*   转轴过**端点**并垂直于杆：$I = \frac{1}{3}mL^2$

### 圆环 / 薄圆筒 (半径 $R$)
*   转轴过**圆心并垂直于环面**：$I = mR^2$
*   转轴沿**直径**：$I = \frac{1}{2}mR^2$

### 圆盘 / 实心圆柱 (半径 $R$)
*   转轴过**中心并垂直于盘面 / 沿圆柱轴线**：$I = \frac{1}{2}mR^2$
*   转轴沿**圆盘直径**：$I = \frac{1}{4}mR^2$

### 球体 (半径 $R$)
*   **薄球壳**（转轴过球心）：$I = \frac{2}{3}mR^2$
*   **实心球**（转轴过球心）：$I = \frac{2}{5}mR^2$

## 3. 关联知识点
- 转动动能: $E_k = \frac{1}{2}I\omega^2$
- 角动量: $L = I\omega$
- 转动定律: $\tau = I\alpha$
