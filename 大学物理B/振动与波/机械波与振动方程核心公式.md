# 简谐振动与机械波核心公式

## 一、 简谐振动方程 (单个质点的“时间”舞蹈)
描述**单一质点**随时间 $t$ 变化的位移：
$$y = A \cos(\omega t + \phi)$$
### 方程内部参数解剖：
- **$A$ (振幅 Amplitude)**: 离开平衡位置的最大距离（决定能量 $E = \frac{1}{2}kA^2$）。
- **$\omega$ (角频率 Angular frequency)**: 每秒转过的弧度数，$\omega = 2\pi \nu = \frac{2\pi}{T}$。它由系统本身的性质决定（如弹簧振子 $\omega = \sqrt{k/m}$，单摆 $\omega = \sqrt{g/l}$）。
- **$\phi$ (初相 Initial phase)**: $t=0$ 时刻质点所处的状态（位置和速度）。通常用**旋转矢量法**判断取值范围。
- **$(\omega t + \phi)$ (相位 Phase)**: 决定质点在任意时刻运动状态的唯一标识。

### 衍生速度与加速度：
- 速度 $v = \frac{dy}{dt} = -A\omega \sin(\omega t + \phi)$
- 加速度 $a = \frac{dv}{dt} = -A\omega^2 \cos(\omega t + \phi) = -\omega^2 y$ *(这是简谐振动的核心判据！)*

---

## 二、 一维平面简谐波方程 (质点群的“时空”接力)
机械波是振动在空间中的传播。波动方程必须同时包含时间 $t$ 和空间坐标 $x$。
如果波沿 $x$ 轴**正方向**传播，且波速为 $u$：
$$y(x,t) = A \cos\left[\omega\left(t - \frac{x}{u}\right) + \phi\right]$$

### 各种炫技的等价变体（极其重要，考卷上全都有）：
1. 引入周期 $T$ 和波长 $\lambda$ ($u = \frac{\lambda}{T}$)：
   $$y(x,t) = A \cos\left[ 2\pi\left(\frac{t}{T} - \frac{x}{\lambda}\right) + \phi \right]$$
2. 引入角波数 $k$ ($k = \frac{2\pi}{\lambda}$)，这是物理系最爱用的形式：
   $$y(x,t) = A \cos(\omega t - kx + \phi)$$
   > ⚠️ **口诀**：“同号向负，异号向正”。括号里 $\omega t$ 和 $kx$ 符号**相反**（一正一负），波向 $x$ **正方向**传播；符号**相同**，波向 $x$ **负方向**传播。

## 三、 驻波方程 (波的“原地踏步”)
两列振幅相同、传播方向相反的相干波叠加，形成驻波。
设正向波 $y_1 = A \cos(\omega t - kx)$，反向波 $y_2 = A \cos(\omega t + kx)$，叠加后：
$$y = 2A \cos(kx) \cos(\omega t)$$
### 驻波解剖：
- **振幅项**：$A(x) = |2A \cos(kx)|$。这说明不同位置的质点，振幅不再相等，而是随位置周期性变化！
- **波腹 (Antinodes)**：振幅最大处，$kx = n\pi$，位置 $x = n \frac{\lambda}{2}$。
- **波节 (Nodes)**：振幅为零处（始终不动），$kx = (n+\frac{1}{2})\pi$，位置 $x = (2n+1) \frac{\lambda}{4}$。

---

## 四、 相位差与波程差 (干涉的核心)
波传播 $x$ 距离，相位就落后 $kx = 2\pi \frac{x}{\lambda}$。
两列波在空间某点相遇，它们的相位差决定了干涉结果（加强还是减弱）：
$$\Delta \Phi = \Delta \phi_0 - \frac{2\pi}{\lambda} (r_2 - r_1)$$
- $\Delta \phi_0$: 两波源的初始相位差。
- $(r_2 - r_1)$: 波程差。
- 如果发生反射，遇到波密介质时会有**半波损失**，即相位突然改变 $\pi$（相当于波程差多走了一半的波长 $\lambda/2$）。
