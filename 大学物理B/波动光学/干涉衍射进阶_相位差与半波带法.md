# 干涉衍射进阶：相位差、光程差与半波带法

## 一、 相位差 ($\Delta \phi$) 与光程差 ($\delta$) 的核心关系
在波动光学中，两束相干光能否产生明纹或暗纹，本质取决于它们相遇时的**相位差**。而相位差往往是由它们走过的**光程差**引起的。

### 1. 核心转换公式
$$\Delta \phi = \frac{2\pi}{\lambda} \delta = k \cdot \delta$$
*   $\Delta \phi$: 相位差 (弧度)
*   $\delta$: 光程差 ($n \cdot \Delta x$)，即折射率乘以几何路程差。
*   $\lambda$: 真空中的波长
*   $k$: 波数 ($k = 2\pi/\lambda$)

**记忆技巧**：光程差每差一个波长 $\lambda$，相位差就差一个完整周期 $2\pi$。因此比例关系为 $\frac{\Delta \phi}{2\pi} = \frac{\delta}{\lambda}$。

### 2. 附加相差 (半波损失)
光从光疏介质射向光密介质，在界面处反射时，会发生 $\pi$ 的相位突变，称为**半波损失**。
*   相位上多了或少了 $\pi$
*   折算到光程差上，相当于多了或少了 $\lambda / 2$。

## 二、 菲涅耳半波带法 (用于分析单缝衍射)
单缝衍射不能用简单的两束光干涉来算，因为缝里有无数个子波。我们采用**半波带法**。

### 1. 核心思想
将单缝处波阵面分割成若干个等宽的窄带，使得相邻两个窄带到达屏幕上某一点 $P$ 时的光程差恰好为 $\lambda / 2$ (即相位差为 $\pi$)。
*   因为相邻半波带到 $P$ 点的光程差为 $\lambda / 2$，所以它们的作用**相互抵消**。

### 2. 划分方法与公式
设单缝宽度为 $a$，衍射角为 $\theta$。单缝上下边缘的光程差为 $\delta = a \sin \theta$。
我们看这个总光程差里包含多少个 $\lambda / 2$：
$$N = \frac{a \sin \theta}{\lambda / 2}$$
*   **$N$ 为偶数 (2k)**：所有的半波带两两抵消，出现**暗纹**。
    *   暗纹条件：$a \sin \theta = 2k \frac{\lambda}{2} = k\lambda \quad (k = \pm1, \pm2...)$
*   **$N$ 为奇数 (2k+1)**：两两抵消后，还剩下一个半波带的光没有抵消，出现**明纹**。
    *   明纹条件：$a \sin \theta = (2k+1) \frac{\lambda}{2} \quad (k = \pm1, \pm2...)$

### 3. 为什么 $k=0$ 不是暗纹？
当 $\theta = 0$ 时，$\delta = 0$。此时无法划分半波带（或者说整个缝就是一个完整的发光面），所有子波到达中心点光程差均为0，发生相长干涉，形成**中央明纹**。这也是为什么单缝衍射暗纹公式中 $k \neq 0$。

## 三、 [[杨氏双缝干涉]]与[[单缝衍射]]对比
| 特性 | 双缝干涉 | 单缝衍射 |
| :--- | :--- | :--- |
| **物理本质** | 分波前，两束相干光叠加 | 同一波前上的无数子波叠加 (半波带法) |
| **光程差 $\delta$** | $d \sin \theta$ | $a \sin \theta$ (上下边缘的光程差) |
| **明纹条件** | $\delta = k\lambda$ | $\delta = \pm(2k+1)\frac{\lambda}{2}$ (中央明纹除外) |
| **暗纹条件** | $\delta = \pm(2k-1)\frac{\lambda}{2}$ | $\delta = k\lambda \quad (k \neq 0)$ |
| **条纹特点** | 等宽、等亮度 | 中央明纹最宽、最亮，向两边逐渐变窄变暗 |
