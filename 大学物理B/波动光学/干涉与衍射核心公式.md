# 波动光学核心公式：干涉与衍射

## 一、 [[杨氏双缝干涉]] (分波前法)
这是光波动说的基石。两束相干光在空间叠加形成明暗相间的条纹。
*   **光程差**：$\delta = r_2 - r_1 = d \sin\theta \approx d \frac{x}{D}$ 
    *(其中 $d$ 是双缝间距，$D$ 是缝到屏的距离，$x$ 是屏上坐标)*
*   **明纹条件**（干涉加强）：$\delta = k\lambda$  ($k = 0, \pm 1, \pm 2...$)
    *   **明纹位置**：$x_k = k \lambda \frac{D}{d}$
*   **暗纹条件**（干涉相消）：$\delta = (2k-1)\frac{\lambda}{2}$
*   **条纹间距**（相邻明纹或暗纹距离）：
    $$\Delta x = \lambda \frac{D}{d}$$
    *(考试最爱考：红光波长长，条纹间距大；蓝光波长短，条纹间距小)*

## 二、 [[薄膜干涉]] (分振幅法)
光在薄膜的上下表面分别反射，两束反射光发生干涉。（典型应用：肥皂泡彩虹、镜头增透膜、劈尖干涉、牛顿环）
*   **核心光程差公式**（光垂直入射或近乎垂直入射时）：
    $$\delta = 2 n d \pm \frac{\lambda}{2}$$
    *(其中 $n$ 为薄膜折射率，$d$ 为薄膜厚度。$\pm \lambda/2$ 是**半波损失**附加项！)*
*   **⚠️半波损失的坑**：
    光从“光疏介质(折射率小)”射向“光密介质(折射率大)”发生反射时，相位突变 $\pi$，光程差要加或减 $\lambda/2$。如果两次反射都有或都没有半波损失，则附加项抵消，公式变为 $\delta = 2nd$。
*   **明纹**：$\delta = k\lambda$ ； **暗纹**：$\delta = (2k-1)\frac{\lambda}{2}$

## 三、 [[单缝衍射]] (夫琅禾费衍射)
衍射和干涉本质相同，但单缝衍射是一条缝里发出的无数个子波相互干涉。
*   **暗纹（极小值）条件**（注意：衍射首先背的是暗纹，这和干涉相反！）：
    $$a \sin\theta = k\lambda \quad (k = \pm 1, \pm 2, \pm 3...)$$
    *(其中 $a$ 是单缝宽度。**绝对不能有 $k=0$**，因为 $k=0$ 是中央明纹！)*
*   **明纹（极大值）位置**（近似）：
    $a \sin\theta = (2k+1)\frac{\lambda}{2} \quad (k = \pm 1, \pm 2...)$
*   **中央明纹线宽度**：
    $$\Delta x_0 = 2 \lambda \frac{D}{a}$$
    *(次级明纹的宽度都是 $\lambda D / a$，只有中央明纹是最宽、最亮的，宽度是其他的 2 倍！)*

## 四、 高级综合考点：缺级现象
当单缝衍射的暗纹恰好落在了双缝干涉的明纹位置上，那个明纹就会因为没有光子而凭空消失！
*   **缺级条件**： $\frac{d}{a} = \frac{k}{k'}$
    (若 $d/a = 3$，则 $k = \pm 3, \pm 6...$ 级的双缝干涉明纹统统消失)。
