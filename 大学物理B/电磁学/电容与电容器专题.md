# 电容与电容器核心考点公式

## 一、 电容的基本定义
电容器容纳电荷本领的物理量。
$$C = \frac{Q}{U}$$
*   $C$: 电容 (法拉, F)
*   $Q$: 极板上所带电荷量的绝对值
*   $U$: 两极板间的电势差

**核心本质**：电容 $C$ 的大小**仅取决于电容器本身的结构（形状、大小、极板间距）和板间的电介质**，与是否带电（$Q$）和两端电压（$U$）**无关**。

---

## 二、 常见电容器电容公式推导与结论
求解电容的通用四步法：
1. 设电容器极板带电 $\pm Q$
2. 用高斯定理求极板间的电场强度 $E$：$\oint \vec{E} \cdot d\vec{S} = \frac{\Sigma q}{\varepsilon}$
3. 用积分求极板间的电势差 $U$：$U = \int_{+}^{-} \vec{E} \cdot d\vec{l}$
4. 套用定义式 $C = \frac{Q}{U}$

### 1. 平行板电容器
*   **公式**：$$C = \frac{\varepsilon_0 \varepsilon_r S}{d}$$
*   $S$ 为极板面积，$d$ 为极板间距，$\varepsilon_r$ 为相对介电常数。

### 2. 圆柱形电容器 (同轴电缆)
由半径为 $R_1$ 和 $R_2$ 的同轴圆柱面组成 ($R_1 < R_2$)，长为 $L$。
*   电场分布：$E = \frac{\lambda}{2\pi \varepsilon r}$ (其中 $\lambda = Q/L$)
*   电势差：$U = \int_{R_1}^{R_2} \frac{Q}{2\pi \varepsilon L r} dr = \frac{Q}{2\pi \varepsilon L} \ln\frac{R_2}{R_1}$
*   **公式**：$$C = \frac{2\pi \varepsilon_0 \varepsilon_r L}{\ln(R_2/R_1)}$$

### 3. 球形电容器
由半径为 $R_1$ 和 $R_2$ 的同心球面组成 ($R_1 < R_2$)。
*   电场分布：$E = \frac{Q}{4\pi \varepsilon r^2}$
*   电势差：$U = \int_{R_1}^{R_2} \frac{Q}{4\pi \varepsilon r^2} dr = \frac{Q}{4\pi \varepsilon} (\frac{1}{R_1} - \frac{1}{R_2})$
*   **公式**：$$C = \frac{4\pi \varepsilon_0 \varepsilon_r R_1 R_2}{R_2 - R_1}$$
*   **孤立导体的电容** (令 $R_2 \to \infty$)：$C = 4\pi \varepsilon_0 R_1$

## 三、 电容器的能量
电容器在充电过程中，电场力做负功，将外力做的功转化为**电场能量**储存在电容器中。
*   **电容器储能公式**：
    $$W_e = \frac{1}{2} C U^2 = \frac{Q^2}{2C} = \frac{1}{2} Q U$$
    *   *(选用哪个公式取决于题目中什么是恒定的。如果接在电源上，$U$恒定，用第一个；如果断开电源，$Q$恒定，用第二个)*

*   **电场能量密度** (单位体积内的电场能量)：
    $$w_e = \frac{1}{2} \varepsilon E^2 = \frac{1}{2} \vec{D} \cdot \vec{E}$$
    *   总能量也可以通过体积分求得：$W_e = \iiint w_e dV$

## 四、 电介质插入电容器的考点陷阱
题目经常考将电介质板或金属板插入电容器极板间，问各物理量如何变化。
**黄金法则（解题核心判断）**：
1. **电源始终连接**：极板间的**电压 $U$ 保持不变**。
2. **充电后断开电源**：极板上的**电荷 $Q$ 保持不变**。

**结论判断**：
插入相对介电常数为 $\varepsilon_r$ 的电介质，总等效于电容 $C$ 增大。
*   若断开电源 ($Q$ 不变)：$C \uparrow \implies U = Q/C \downarrow \implies E = U/d \downarrow \implies W_e = Q^2/2C \downarrow$ (电场力做正功吸入介质)。
*   若连接电源 ($U$ 不变)：$C \uparrow \implies Q = CU \uparrow \implies E = U/d$ 不变 $\implies W_e = \frac{1}{2}CU^2 \uparrow$ (电源做功)。
