# SSRF 解题 SOP 清单

## Step 1：确认可控点
- [ ] 找到 URL/路径相关参数
- [ ] 替换为自有域名，确认服务端是否请求

## Step 2：确认反馈通道
- [ ] 有直显 / 半盲 / 全盲
- [ ] 记录状态码、长度、耗时基线

## Step 3：基础目标测试
- [ ] `127.0.0.1`
- [ ] `localhost`
- [ ] `169.254.169.254`
- [ ] 常见内网端口

## Step 4：限制识别
- [ ] 协议限制？
- [ ] host 黑名单/白名单？
- [ ] 是否跟随重定向？
- [ ] 是否存在 WAF 关键字过滤？

## Step 5：绕过执行
- [ ] IP 多表示法
- [ ] URL 编码/双编码
- [ ] `@`、`#`、大小写、末尾点
- [ ] 重定向中转

## Step 6：目标优先级攻击
- [ ] 内网管理页面
- [ ] 云元数据
- [ ] Redis/MySQL 等内网服务

## Step 7：证据与复盘
- [ ] 保存 payload 与响应差异
- [ ] 写入 [[14-练习清单与复盘模板]]
