# Lab 02：Jinja2 对象遍历

## 练习目标
- 从基础表达式执行走到对象探测
- 学会从上下文对象继续扩展利用链

## 场景描述
- 题目已经可以确认 `{{7*7}}` 生效
- 你的任务不是只输出数字，而是找到能帮助你读文件的对象

## 推荐步骤
1. 先测试 `{{config}}`
2. 再测试 `{{request}}`
3. 测试 `{{url_for}}` 和 `{{get_flashed_messages}}`
4. 尝试访问函数相关全局信息
5. 记录哪些对象可见、哪些对象不可见

## 关键训练点
- 不是盲目复制长链，而是一步一步确认：
  - 有没有这个对象
  - 这个对象是不是函数
  - 能不能拿到全局字典
  - 全局字典里有没有你要的能力

## 预期收获
- 能理解为什么函数对象是关键跳板
- 能把“入口对象 -> 全局 -> 危险能力”串起来

## 复盘问题
- 哪个对象最先给了你突破口？
- 你走的是函数链还是类层级链？
- 哪一步如果失败，你的备选方案是什么？
