#过滤空格
# ctfshow_web入门_178
与[[ctfshow173sql修改输出结果]]一样是加入了过滤，但是过滤更隐蔽，且过滤内容较多。
![[Pasted image 20260408140451.png]]
先对它进行过滤检查，发现：
- 过滤了空格。
- -无法正常使用，#无法正常使用，但这两者似乎并不是被过滤，而是解析出错。
	- \#可能是被拼接到url后当成段标记符了，所以用%23来对\#进行编码
	- -不知为何无法使用
所以构造payload:
- `'or(1=1)%23`
	- `()`在进行表达式运算时可以省略掉空格
- `or'1'='1'%23`
- or/\*\*/1=1/\*\*/%23
- .....
其实只需要专注于避开空格。
一些空字符可以用来辅助避开空格：
- %0a换行符
- %0b 垂直制表符
- %0c 换页
- %0d 回车
- %09 tab
一些符号可以帮助省略空格：
- `()`除了不能与关键字连上，其余的几乎可以
- `'`可以与关键字连上，也能和大部分其他的连
