#排查被过滤字符 可以用普通文本字符来将疑似被过滤的字符进行包裹，比如：
```
A<xss>B
```
这样的话如果 `xss` 被过滤，那么显示可能为空; 如果没有被过滤，那么应该会显示 `AB`。
- 不过有时候可能也会分段过滤，这时候可以构造比如 `A<B>C[D]E{F}G/H\I` 来进行检测。

另外我在本题中发现有时候onload必须带引号，这种情况通常发生于属性值过早遇到终止符(空格或者引号等)。比如 `A<iframe/onload=alert("xss")/>B` ，在这种情况下，当 `onload` 想要读取属性，它会过早遇到终止符 `"` （或是解析异常），导致属性读取不成功。
当改成 A<iframe/onload=alert(\`xss\`)/>B时就可以正常使用啦。当然对于数字来讲也可以完全不用引号。 

### 裸值的终止符完整列表

|字符|说明|
|---|---|
|空格/Tab/换行|最常见的终止符|
|`>`|标签结束|
|`"`|触发解析异常|
|`'`|触发解析异常|
|`` ` ``|部分解析器终止|
