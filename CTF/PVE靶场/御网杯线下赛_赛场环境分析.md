# 御网杯线下精英赛 赛场环境分析

根据《第十届“御网杯”网络安全大赛_线下精英赛_参赛指南》，赛场网络及设备环境如下：

## 1. 官方提供设备与配置
- **网络设备**：每组配置一台路由器（1个WAN口，3个LAN口）。
- **电源**：每工位配备 220V 电源。
- **特殊环境（极度重要）**：赛场部署有**全场信号屏蔽仪**，切断一切无线网络和蓝牙。

## 2. 选手必须自备的物料
- **参赛终端**：仅允许使用笔记本电脑（**严禁携带台式机、小型服务器**）。
- **网络线缆**：每组必须自带 **3条 2-3米长的网线（跳线）**。
- **外设**：由于有信号屏蔽仪，**必须带**有线键盘、有线鼠标。
- **网卡**：无内置网口的轻薄本/Mac，必须提前买好有线网口扩展坞（USB/Type-C转RJ45）。

## 3. 网络连通机制
- **IP分配**：电脑网卡必须设置为“自动获取 IP 地址（DHCP）”。
- **竞赛访问**：本地局域网仅能访问平台，必须通过提前下载安装的 **OpenVPN** 客户端（执行 `openvpn xxx.ovpn`）拨入靶机内网。

## 4. 关于外带交换机的分析与风险
官方规则中**未明文禁止**携带小型交换机（仅明文禁止了台式机和小型服务器），但由于官方路由器刚好提供 3 个 LAN 口（对应队伍的 3 名成员），且明确要求自带 3 根网线，其设计初衷是**一人直连一孔**。
如果执意使用外带交换机（如上一篇笔记提到的高速组网方案），可能会面临以下技术深坑：
1. **端口安全（Port Security）限制**：如果官方路由器开启了 MAC 地址数量限制（一个端口只允许一个 MAC），你的交换机接上去会导致其他队员的流量被直接阻断（Drop）。
2. **DHCP 泛洪被封禁**：如果私接的交换机或旁路由配置不当，向官方网络发送了非法的 DHCP Offer 或广播风暴，会被赛场安全设备判定为恶意攻击，直接取消成绩。
建议方案：带上备用，现场先尝试直连，如果局域网大模型确实卡顿，再去与当值裁判沟通确认是否能接入自己的傻瓜交换机。
