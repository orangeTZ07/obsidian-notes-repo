# 二叉排序树 (Binary Sort Tree / BST)

## 核心定义（一句话口诀）
**“左小、右大、根中间”**

它是一棵二叉树，必须满足以下极其严格的家族族规：
1. 如果它有**左子树**，那么**左子树上所有结点**的值，必须**统统小于**根结点的值。
2. 如果它有**右子树**，那么**右子树上所有结点**的值，必须**统统大于**根结点的值。
3. 它的左、右子树内部，也必须分别遵守这条“左小右大”的族规。

*(注：在严蔚敏版教材的基础题中，通常默认树里没有两个相等的数字)*

## 形象化理解（结合折半查找）
二叉排序树，本质上就是把我们前面做的**“折半查找”的过程，用树枝给画了出来**！
当你站在任何一个路口（结点）找数字时：
- 目标比脚下的数字**小**？毫不犹豫往**左**拐！
- 目标比脚下的数字**大**？毫不犹豫往**右**拐！

## 🌟 必考终极考点：中序遍历
如果你对一棵二叉排序树进行**中序遍历**（按照“左子树 -> 根结点 -> 右子树”的顺序踩点），把路过的数字依次写下来，你会发现一个奇迹：
**它变成了一个从小到大完全排好序的有序序列！**

*(因为中序遍历永远是先输出较小的左边，再输出中间，最后输出较大的右边。)*
