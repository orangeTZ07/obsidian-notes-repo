# 堆排序：建初堆

在严蔚敏版《数据结构》中，堆排序是非常重要的一种选择排序。

## 1. 什么是堆？

堆可以看作是一棵**完全二叉树**的顺序存储（用数组存）。
对于大根堆（大顶堆），必须满足：**任何一个非叶子节点的值，都大于或等于它的左孩子和右孩子的值。**

假设数组索引从 $1$ 到 $n$：
- 节点 $i$ 的左孩子是 $2i$
- 节点 $i$ 的右孩子是 $2i + 1$
- 节点 $i$ 的父节点是 $\lfloor i / 2 \rfloor$

## 2. 如何建初堆？

建初堆的核心思想是：**从最后一个非叶子节点开始，逆序往上，依次对每个节点进行“下沉”调整（Sift Down / HeapAdjust）。**

**步骤：**
1. 找到最后一个非叶子节点的位置：$i = \lfloor n / 2 \rfloor$。（$n$ 为元素总个数）
2. 检查节点 $i$ 和它的孩子们：
   - 找出孩子中最大的一个。
   - 如果最大的孩子比节点 $i$ 大，那么它们交换位置。
3. **注意：** 交换之后，原来的节点 $i$ 会落到下一层，可能会破坏下一层的堆结构！因此，必须**继续往下检查**，直到它比所有的孩子都大，或者它变成了叶子节点为止。这叫做一次完整的“下沉”。
4. $i$ 减 $1$，继续对前一个非叶子节点做相同操作，直到根节点（$i = 1$）调整完毕。

## 3. 堆排序的过程
1. **建初堆**：将无序序列调整成大根堆。此时最大元素在堆顶。
2. **交换与筛选**：
   - 将堆顶元素（最大的数）与堆尾元素交换，最大的数就排好位置了。
   - 堆的长度减1，对新的堆顶元素进行一次“下沉”调整，恢复堆结构。
   - 重复此过程，直到堆中只剩1个元素。
